About

This is a web version of the LLM Chess Engine research project. Play as White — the engine plays Black. Set LLM_ENGINE_ENABLED=1 and provide the VICE engine binary to enable Llama-3 moves. Otherwise, a random move fallback is used.

Research Question: Can constraining an LLM with a legal-move list eliminate hallucinated chess moves?
